## Run Locally

Clone the project

```bash
  git clone git@github.com:vedanta28/msb-academy.git
```

Go to the api directory

```bash
  cd api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Go to the client directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
